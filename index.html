Report

"Before designing my website, I did some research into the web standards used. I referenced sites such as Facebook, Google, Amazon and considered what they have in common with one another. One such standard is a navigation bar that is placed at the top of each page which allows users to easily navigate through the site. Another is the consistent colour scheme, font and style throughout each page which allows for better readability and keeps the website clean."

"When designing my website, I considered these standards – putting a navbar at the top of each page and using the same colours and styles."

"There are also standards put in place by the W3C to enforce conformity between browsers. These are standards such as HTML 5 which provides new elements to all for more complex layouts. Tags such as “section” allow for better readability, making writing and interpreting HMTL and CSS much easier. HTML 5 also removed reliance on third party plug-ins – making websites less demanding and ultimately quicker and more secure."

"I utilised these new standards affectively – using the section tag for appropriate “divs”, a video tag for a background video and canvas for my customiser."
